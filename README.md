# number-guessing-game

## Description
A Python command-line game where the computer randomly selects a number between 1 and 10, and the user attempts to guess it. The program provides feedback if the guess is too high or too low and continues until the correct number is gussed

## Features
-Random number generation
-User input validation
-Feedback for incorrect guesses
-Replay functionality

## Technologies Used
-Python
-Loops
-Conditional Logic
-Exception handling
-'random' module
