# number-guessing-game

## Description
A Python command-line game where the user tries to guess a randomly generated number between 1 and 10.
The progarm gives feedback if the guess is too high or too low and continous until the correct number is guessed.
Users can play multiple rounds.

## Features
- Random number generation using Python 'random' module
- User input validation
- Feedback for guesses (too high, too low, correct)
- Replay functionality for multiple rounds
- Menu-style interaction for smooth gameplay

## Technologies Used
- Python 3
- Standard library only ('random' module)


## How to Run
1. Clone the repository
2. Navigate to the project folder
3. Run the progam:
   python number-guessing-game.py

## Example Usage
- User is promted to enter a number between 1 and 10
- Progam outputs feedback: "Too high..." or "Too low..."
- When guessed correctly, the progam congratulates the user
- User is asked if they want to play again
